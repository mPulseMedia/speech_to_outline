## speech_to_outline

Real-time “speech → outline” organizer demo.

### Run

```bash
npm install
npm run dev
```

Then open the local URL Vite prints.

### Single-file webpage (no build)

Open `speech-to-outline.html`.

Note: for **mic** permissions / speech recognition, browsers generally require `http://localhost` or `https://` (not `file://`).
